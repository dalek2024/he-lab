# He Lab 网站维护与更新说明

本文档说明如何维护和更新 He Lab 实验室网站的各项内容。

---

## 一、数据来源概览

| 内容 | 数据来源 | 更新方式 |
|------|----------|----------|
| 论文 Publications | Google Sheets | 编辑表格，自动同步 |

## 二、论文（Publications）

### 2.1 数据源

论文数据来自 **Google Sheets**，通过 CSV 格式动态加载。

- **配置文件**：`data.js` 中的 `PUBLICATIONS_CSV_URL`
- **展示页面**：`publications.html`（全部论文）、首页（精选论文）

### 2.2 Google Sheet 表头格式

| 列名 | 说明 | 示例 |
|------|------|------|
| Year | 发表年份 | 2024 |
| Title | 论文标题 | Molecular mechanisms of... |
| Authors | 作者列表 | Zhang L, Wang Y, He Y, et al. |
| Journal | 期刊名称 | Nature Communications |
| Link | 论文链接（PDF 或 DOI） | https://doi.org/... |
| Featured | 是否在首页精选展示 | yes / no |

### 2.3 更新步骤

1. 打开对应的 Google Sheet
2. 新增或修改行数据
3. 需要首页精选展示的论文，在 **Featured** 列填写 `yes`，否则填 `no` 或留空
4. 若更换了表格，需在 `data.js` 中更新 `PUBLICATIONS_CSV_URL`

### 2.4 发布 CSV 链接

若使用新的 Google Sheet，需发布为 CSV：

1. 打开 Google Sheets → **文件** → **共享** → **发布到网页**
2. 选择 **逗号分隔值 (.csv)**
3. 复制生成的链接，粘贴到 `data.js` 的 `PUBLICATIONS_CSV_URL`

## 三、团队成员（Team）

### 3.1 数据源

团队成员数据来自 **Google Sheets**。

- **配置文件**：`data.js` 中的 `TEAM_CSV_URL`
- **展示页面**：`team.html`、首页 Team 区域

### 3.2 Google Sheet 表头格式

| 列名 | 说明 | 示例 |
|------|------|------|
| Name | 姓名 | Zhang San |
| Role | 职位/角色 | PhD Student |
| Program | 入学年份等 | 2023 |
| Research | 研究方向简介 | Antibiotic Resistance |
| Photo | 照片链接（可选） | https://.../photo.jpg |
| Status | 在职状态 | Current / Alumni |

### 3.3 状态说明

- **Current**：在职成员，显示在「Current Members」
- **Alumni** 或其他：已毕业/离组，显示在「Alumni」

### 3.4 更新步骤

1. 打开对应的 Google Sheet
2. 新增、修改或删除成员行
3. 无照片时 Photo 可留空，系统会显示首字母占位图
4. 更换表格时，在 `data.js` 中更新 `TEAM_CSV_URL`

## 四、研究方向（Research）

### 4.1 修改位置

编辑 `data.js` 中的 `RESEARCH_AREAS` 数组。

### 4.2 格式示例

```javascript
{
  title: "研究方向标题",
  description: "详细描述文字...",
  icon: "fa-shield-halved"  // FontAwesome 图标类名
}
```

### 4.3 常用图标

可在 [Font Awesome](https://fontawesome.com/icons) 搜索图标，使用 `fa-xxx` 格式，如：`fa-dna`、`fa-atom`、`fa-virus`、`fa-flask`。

## 五、内部链接（Internal）

### 5.1 修改位置

编辑 `data.js` 中的 `INTERNAL_LINKS` 数组。

### 5.2 格式示例

```javascript
{
  title: "链接标题",
  url: "https://...",      // 实际链接
  description: "简短说明"   // 可选
}
```

## 六、日历（Calendar）

### 6.1 当前实现

日历已直接嵌入 `calendar.html`，无需依赖 `data.js`。

### 6.2 更换日历

如需更换为其他 Google 日历：

1. 打开 Google Calendar → **设置** → 选择日历 → **集成日历**
2. 复制「嵌入代码」中 iframe 的 `src` 属性值
3. 在 `calendar.html` 中找到 iframe，将 `src="..."` 替换为新链接
4. 同时更新下方「Open in Google Calendar」链接（格式：`https://calendar.google.com/calendar/u/0?cid=日历ID`）

## 七、首页静态内容

### 7.1 About Us（关于我们）

编辑 `index.html`，搜索 `id="about"`，修改其中的 `<p>` 段落文字。

### 7.2 Principal Investigator（PI 简介）

编辑 `index.html`，搜索 `id="pi"`：

- 更换照片：修改 `<img src="...">` 的 `src` 地址
- 修改简介：编辑下方 `<p>` 段落

### 7.3 联系方式

各页脚的联系方式（邮箱、电话、地址）分散在各 HTML 文件中，需分别修改。可搜索 `heyx@lzu.edu.cn` 或 `Contact` 定位。

## 九、常见问题

### Q: 修改 Google Sheet 后网页未更新？

- 发布后的 CSV 有缓存，通常几分钟内会更新
- 可尝试强制刷新页面（Ctrl+Shift+R）

### Q: 论文/团队显示「加载失败」？

- 检查网络连接
- 确认使用 http/https 访问，避免直接用 `file://` 打开
- 建议用本地服务器：`npx serve .` 或 `python -m http.server 8000`

### Q: 如何更换 Google Sheet 链接？

在 `data.js` 中修改 `PUBLICATIONS_CSV_URL` 或 `TEAM_CSV_URL`，确保新表格表头格式与本文档一致。
